## eye-onic

Clone this repo to your machine and cd into the app directory:<br>
`cd medhacks/ionic-react-app`

If you don't have ionic CLI installed:<br>
`npm install -g @ionic/cli`

Install dependencies:<br>
`npm install`

And serve!<br>
`ionic serve`
